# Army of the Damned Campaign Pack

This pack adapts the adventure source in `armyofthedamned.pdf` into the
same campaign-pack structure used by `campaigns/morgana/`.

## Included

- `campaign.yaml`
- `chapters/chapter_01` through `chapters/chapter_05`
- `locations/` for the road, Shadowgrange, the farbog, and Mauer Estate
- `triggers/` using the current built-in trigger DSL
- `world/` lore, factions, and timeline
- `characters/seed/` with a starter player character
- `prompts/` compatible with the current narrator pipeline

## Scope

This is a structured, runtime-friendly paraphrase of the scenario rather than a
page-by-page transcription of the PDF.
